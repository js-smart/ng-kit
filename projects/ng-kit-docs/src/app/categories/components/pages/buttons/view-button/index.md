# View Button

A button for view or details actions, often with an eye icon.

## Description
The View Button extends the Base Button and is used to trigger view or details actions. It is styled with an eye icon and is accessible.

## API
- **disabled**: `boolean` — Disables the button.
- **ariaLabel**: `string` — Accessibility label.
- **(click)**: `EventEmitter<MouseEvent>` — Emits when the button is clicked.

## Usage
This can not be used alone. You must use extended button components
