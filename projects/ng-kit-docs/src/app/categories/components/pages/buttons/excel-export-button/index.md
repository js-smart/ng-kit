A button for exporting data to Excel format. The Excel Export Button is a standalone button component styled for Excel export functionality with a dark green background.

## Usage

```html group="usage" name="Directive (Preferred)" active
<button excelExportButton>Excel</button>
```

```html group="usage" name="Component"
<excel-export-button></excel-export-button>
```

## API Reference
The Excel Export Button is a standalone component and does not extend the Base Button. It is a simple button with fixed styling and text.

This component does not expose any configurable inputs or outputs.
